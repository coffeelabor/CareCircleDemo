# My Care Circle

Static GitHub Pages proof of concept for a content-first community care support site.

## What this POC includes

- Home, About, Support Ideas, Articles, and Resources pages
- Ten starter article pages
- No build step or package install
- No forms, accounts, comments, registries, payments, or personal/health data collection

## Local preview

Open `index.html` directly in a browser.

## GitHub Pages deployment

1. Push `master` to GitHub.
2. Open the repository on GitHub.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `master` and `/ (root)`.
6. Save and wait for GitHub to publish the Pages URL.

Use the standard GitHub Pages URL first. Add `mycarecircle.org` later after DNS access is available.
